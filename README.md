# Integração com PokeAPI em python

Projeto desenvolvido para demonstrar habilidades de backend e consumo de APIs REST.


## Funcionalidades
- Consumo de dados em tempo real da PokeAPI
- Tratamento de erros para entradas inválidas (Pokémon não encontrado)
- Manipulação e formatação de dados JSON co suporte a caracteres especiais (UTF-8).

## Tecnologias
- Biblioteca Requests para comunicação HTTP
- Biblioteca JSON para serialização e formatação de dados

# Como rodar o projeto
1. Instale a biblioteca necessária:
    `pip install requests`
2. Execute o script
    `python main.py`
